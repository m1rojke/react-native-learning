# React and React Native Learning Course

## О курсе
75-дневный курс от JavaScript основ до полноценного портфолио с React и React Native.

## Структура курса

### День 1: Arrow Functions и Template Literals
- Стрелочные функции
- Template Literals
- Практические упражнения
- Домашнее: Создание визитной карточки

### День 2: Деструктуризация и операторы
- Let и Const
- Деструктуризация объектов и массивов
- Spread оператор
- Rest оператор

### Дни 3-4: Методы массивов
- map() - преобразование
- filter() - фильтрация
- reduce() - свёртка
- find() - поиск
- Комбинирование методов

### День 5-6: Git и GitHub
- Инициализация репозитория
- Основные команды (add, commit, push, pull)
- Ветки и слияние
- GitHub репозиторий

## Проекты в портфолио
1. GitHub Portfolio (дни 5-8)
2. Todo List App (React, дни 14-16)
3. Weather App (React, дни 17-19)
4. Personal Portfolio (React, дни 23-25)
5. E-commerce Catalog (React, дни 26-31)
... и ещё 10 проектов

## Запуск примеров
