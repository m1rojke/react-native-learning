# Git Workflow Guide

## Основной процесс
1. `git switch -c feature/name` - создание ветки
2. Make changes
3. `git add .` - staging
4. `git commit -m "message"` - коммит
5. `git push -u origin feature/name` - загрузка
6. Create Pull Request on GitHub
7. `git merge` - слияние
8. `git push` - загрузка на GitHub

## Команды
- `git status` - статус
- `git log --oneline` - история
- `git diff` - изменения
